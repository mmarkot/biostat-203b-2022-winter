    A more thorough book treatment of the practical imputation strategies is the book Flexible Imputation of Missing Data by Stef van Buuren.

  2. Explain the jargon MCAR, MAR, and MNAR.

Solution: MCAR refers to data missing completely at random. This means that there is no pattern to the data that is missing and there is no difference between observtions with missing and non-missing data. If diagnostic plots differ between the imputed and non-missing data, this can mean that MCAR does not apply. MAR means missing at random. This is data that is missing correlatedly with an observed variable. In this case there is information as to why the data is missing for certain observations. MNAR mean missing not at random. This category explains cases where there is a pattern to the missing data but it does not fit the MAR definition because the explanation of the pattern is unobserved. Essentially the correlatedness of the missing data is clear but the cause is unknown.

  1. Explain in a couple of sentences how the Multiple Imputation by Chained Equations (MICE) work.

Solution: MICE iteratively imputes data for missing values going variable by variable using all other variables. Missing data is first imputed randomly and then the full data for all columns but one are used to predict values for that one with random forest. This is repeated for all variables, and if necessary the entire process can be repeated if the values have not converged.
